CSS-специфічність визначається чотирма категоріями:

  1. Inline-стилі (style=“color: red;”) (A): Рівень 1:0:0:0.
  2. Ідентифікатори (#id) (B): Рівень 0:1:0:0.
  3. Класи, атрибути, псевдокласи ( .class, [attr=value], :hover ) (C): Рівень 0:0:1:0.
  4. Типи елементів, псевдоелементи ( h1, ::before ) (D): Рівень 0:0:0:1
  5. Коли при оголошенні стилю використовується модифікатор,
    !important це оголошення отримує найвищий пріоритет серед усіх інших оголошень !important
