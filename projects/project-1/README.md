# About
The website is templated with Figma. 
+ Edit chart`s values\
+ Edit chart`s labels\
+ Edit chart`s colors\
+ Swap positions of data in charts\
+ open/close sidebar\
+ switch themes(light/dark)\
+ Media queries and adaptive layout\

Created with mobile-first method\

+ chart.js\
+ SASS\
